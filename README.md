## ğŸš€ Getting Started

### Prerequisites

Before starting this learning journey, make sure you have:

- **Node.js** (version 18 or higher)
- **npm** or **pnpm** package manager
- A code editor (VS Code recommended)
- Basic knowledge of HTML, CSS, and JavaScript
- Familiarity with ES6+ JavaScript features

### Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/deniswachira/Learning-ReactJs101Series-Repo.git
   cd Learning-ReactJs101Series-Repo
   ```

2. **Navigate to any lesson folder** and follow the specific README instructions for that lesson.

3. **Install dependencies** for each project:
   ```bash
   cd 0-Intro-ReactJS
   pnpm install
   ```

4. **Start the development server**:
   ```bash
   pnpm dev
   ```
